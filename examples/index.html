favicon-env

Tint your favicon per environment, so you can tell instances apart at a glance.

$ pnpm add favicon-env GitHub npm
favicon-env — demo ← this favicon (and your real browser tab) updates when you pick a type

Each card is a live envFavicon(...) config. Click one to apply it.

Zero dependencies · runtime + build-time (SSR) · MIT. Full API in the README.
